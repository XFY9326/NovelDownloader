# NovelDownloader

简单的小说下载爬虫

## 特性

- 清晰代码框架
- 多线程下载
- 自动错误重试

## 目前支持

- http://www.lanlanguoji.com/
  - /xiaoshuo/<novel_id>/
- https://www.trxs.cc/
  - /tongren/<novel_id>.html
- https://www.xiashuyun.com/
  - /<novel_id>/chapters.html
- http://www.buxia.net/
  - /xiaoshuo/<novel_id>/
- http://www.quanben.io/
  - /n/<novel_id>/list.html
- https://www.aixdzs.com/
  - /novel/<novel_id>
